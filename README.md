Food Delivery Frontend Application
Live Demo: https://food-delivery-frontend-pied.vercel.app/

Project Overview
A responsive frontend-only food delivery web application designed to simulate a modern restaurant ordering interface. The project focuses on clean UI architecture, structured layout design, and professional deployment workflow.

Tech Stack

HTML5
CSS3 (Flexbox & Grid)
JavaScript
Git & GitHub
Vercel (Deployment)

Key Features

Sticky navigation bar
Hero section with call-to-action
Category-based food segmentation
Product grid with pricing and ratings
Add to Cart UI interaction
Responsive design (mobile + desktop)
Clean, consistent color theme

Deployment

The project is deployed via Vercel with continuous deployment enabled.
Every push to the main branch automatically triggers a new production build.

Skills Demonstrated

Frontend development fundamentals
Responsive UI engineering
Component-based layout structuring
Version control workflow
CI/CD deployment process
